# Terminal AI Chatbot

## Features
- Groq Integration
- Gemini Integration
- Runtime Model Switching
- Automatic Fallback System
- Error Handling
- Terminal Interface
- Streaming Responses (Bonus)

## Installation

npm install

## Setup

Create .env

GROQ_API_KEY=your_key
GEMINI_API_KEY=your_key

## Run

npm start

## Commands

/groq
/gemini
/help
/clear
/exit

## Project Structure

services/
utils/
index.js
